Circuit-Code Description

Since each bulb is in parallel (same voltage and equal current all around), I use 220 ohm resistors for each bulb.

The white, green, and red bulbs are linked to ports 11, 12, and 13 respectively. Going in reverse numerical order, each bulb will power on for half a second and power off in the reverse of the first sequence for half a second.

Code

// Setup light timer to Half a Second int timer = 500; void setup() { // Initialize Pins 11-13 for LEDs pinMode(13, OUTPUT); pinMode(12, OUTPUT); pinMode(11, OUTPUT); } void loop() { //Loop through pins 11-13 to turn them on for half a second for(int pinNum = 13; pinNum >= 11; pinNum--) { digitalWrite(pinNum, HIGH); delay(timer); } //Loop through pins 11-13 in reverse order to turn them off for half a second for(int pinNum = 11; pinNum <= 13; pinNum++) { digitalWrite(pinNum, LOW); delay(timer); } }

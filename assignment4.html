Since the LCD requires a resistance of 0-20kΩ, I decided to use the 10kΩ Potentiometer, giving a range of 50Ω to 10kΩ (based on the datasheet).

From that, the current of that system ranges from 0.5mA to 0.1A

To power the LED backlight, I used a 220Ω resistor to keep the current at 20mA from the 5V power source.

Circuit-Code Description

The interaction with the circuit is very simple. Using the DHT11 Temperature and Humidity sensor and the LiquidCrystal Display, I created a circuit that reads the temperature and humidity.

After a second, the sensor will read a new temperature and humidity and display each value on their own line on the LCD.

Code

//Import LiquidCrystal and DHT Libraries #include #include //Set Temp/Humidity Sensor to Pin 7 dht DHT; #define DHT11_PIN 7 //Initialize LiquidCrystal Display to respective pins LiquidCrystal lcd(12, 11, 5, 4, 3, 2); void setup() { //Set crystal display to 2 rows of 16 character lengths lcd.begin(16, 2); } void loop() { //clear LiquidCrystal display from last time lcd.clear(); //Set Cursor to top left corner lcd.setCursor(0, 0); //Read Temperature and Humidity from Sensor int chk = DHT.read11(DHT11_PIN); //Print Temperature Reading on first row lcd.print("Temp = "); lcd.print(DHT.temperature); lcd.setCursor(0, 1); //Print Humidity Reading on second row lcd.print("Hum = "); lcd.print(DHT.humidity); delay(1000); }
